# Data Directory

The data directory contains all data files generated by the chatPsych application.

### User Interaction Data
- `interactions.json` - Complete user interaction data in JSON format
- `interactions_backup.csv` - Backup of interaction data in CSV format

### Survey Data (Separated by Type)
- `pre_survey.json` - Pre-survey responses in JSON format
- `pre_survey.csv` - Pre-survey responses in CSV format
- `post_survey.json` - Post-survey responses in JSON format
- `post_survey.csv` - Post-survey responses in CSV format
- `survey_config.json` - Survey configuration settings

### Legacy Survey Data (Mixed Pre/Post - Deprecated)
- `survey.json` - Legacy mixed survey responses in JSON format (no longer updated)
- `survey.csv` - Legacy mixed survey responses in CSV format (no longer updated)

### Popup Data
- `popup.json` - Post-chat popup selections in JSON format
- `popup.csv` - Post-chat popup selections in CSV format

### System Data
- `download_log.json` - Log of file downloads from the research dashboard

## Data Structure Changes

As of the latest update, survey data is now logged to separate files based on survey type:
- **Pre-surveys**: Data is logged to `pre_survey.json` and `pre_survey.csv`
- **Post-surveys**: Data is logged to `post_survey.json` and `post_survey.csv`

This separation ensures that pre and post survey data with different structures and column names don't interfere with each other, making data analysis much cleaner and more reliable.

The legacy `survey.json` and `survey.csv` files are kept for backward compatibility but are no longer updated with new survey responses.

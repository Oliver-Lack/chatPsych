About

The chatPsych Interface is designed for human-AI interaction research.
This platform is open-source and available for research adaptions at chatPsych

Don't like coding? Use this dashboard to create your human-AI interaction experiment

First, setup the following:
-> Necessary API keys need to be set in a .env file of chatPsych's working directory
-> Configure redirection links, timer and finish button, agents, and survey deployment
-> Configure manual interactive buttons and hardcoded prompts


-> An editor for manual interaction buttons and hard coded prompts.
-> Survey Deployment, linked to redirection links
-> Text-Audio API integration, settings in dashboard
-> Test-Audio Graphic, linked to redirection links

For the newest version of the interface visit the public repo:
